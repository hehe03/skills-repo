# 欠料归因分析规则

本文档描述欠料归因分析的L2分类规则。

## 规则说明

### 1. 网容异常

- **条件1**: 网容 > 0
- **匹配**: 条件1不满足时匹配（即网容 <= 0 或为空）

### 2. 用量异常

- **条件1**: M2 > M3-M13最大值
- **条件2**: 历史交易数量 > 原始基线
- **匹配**: 条件1或条件2满足

### 3. 补库异常

- **条件1**: 历史交易数量 > 补库提前期补库和调拨汇总数量
- **匹配**: 条件1满足

### 4. 基线异常

- **条件1**: 修改基线ROP < 推荐基线（修改基线ROP存在时）
- **条件2**: 历史交易数量 > 原始基线
- **条件3**: 原始基线=0 且 历史交易数量=0 且 网容>0
- **条件4**: 历史交易数量=0 且 M3-M13最大值=0 且 网容>0 且 推荐基线 > 原始基线
- **匹配**: 条件1或条件2或条件3或条件4满足

### 5. 计划参数异常

- **条件1**: 最终补库提前期 < 计算补库提前期
- **匹配**: 条件1满足

### 6. 补库供应不及时

- **条件1**: 补库在途时间（天） > 最终补库提前期
- **条件2**: 调拨在途时间（天） > 最终补库提前期
- **匹配**: 条件1或条件2满足
- **注意**: 补库在途时间、调拨在途时间格式为"数字d"（天）

### 7. 责任库房异常

- **条件1**: 责任库房预测物流时长（天） > SLA承诺时间（天）
- **匹配**: 条件1满足
- **注意**: 责任库房预测物流时长、SLA承诺时间格式为"数字h"（小时）

### 8. 替代交付异常

- **条件1**: 单一替代库存汇总 < 总欠料数量
- **条件2**: 组合替代关系非空 且 (组合替代库存汇总为空/nan 或 任一组合替代物料库存小于对应需求)
- **匹配**: 条件1且条件2都满足
- **注意**: 当组合替代关系为空[]时，表示没有组合替代方案，不标记为替代交付异常

## Trace 要求

分析脚本应输出逐规则 trace，至少包含：

1. 规则是否命中；
2. 每个条件的布尔值；
3. 关键字段原值与必要的解析值；
4. 匹配原因说明。
