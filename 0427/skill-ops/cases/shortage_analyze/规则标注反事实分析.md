# 规则/标注反事实对照分析

## 分析对象

- 目标标签：替代交付异常
- 触发冲突的典型条件：`组合替代关系=[]` 且 `单一替代库存汇总 < 总欠料数量`
- 基线一致样本数：126
- 基线一致率：75.45%
- 满足“若放宽规则则会新增标签”的样本数：116
- 其中当前标注包含目标标签、预测不包含目标标签的直接冲突样本数：29
- `组合替代关系=[]` 且标注包含目标标签的样本数：29

## 方案一：修改规则

- 假设：只要 `组合替代关系=[]` 且单一替代库存不足，也判为 `替代交付异常`。
- 一致样本数将变为：74
- 一致率将变为：44.31%
- 新增修复的样本数：28
- 新增破坏的一致样本数：80

## 方案二：修改标注

- 假设：对 `组合替代关系=[]` 的样本，从标注中移除 `替代交付异常`。
- 一致样本数将变为：154
- 一致率将变为：92.22%
- 新增修复的样本数：28
- 新增破坏的一致样本数：0

## 推断

- 更可能是标注与规则文档冲突，而不是规则本身需要放宽。
- 主要原因是：放宽规则只修复 28 个冲突样本，却会额外破坏 80 个原本一致的样本；相反，修改标注同样修复 28 个冲突样本，且不会破坏已有一致样本。
