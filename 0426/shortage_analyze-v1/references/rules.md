# 欠料归因分析规则

本文档描述欠料归因分析的L2分类规则。

## 规则说明

### 1. 网容异常

- **条件1**: 网容 > 0
- **匹配**: 条件1不满足时匹配（即网容 <= 0 或为空）

### 2. 用量异常

- **条件1**: M2 > M3-M13最大值
- **条件2**: 历史交易数量 > 原始基线
- **匹配**: 条件1或条件2满足

### 3. 补库异常

- **条件1**: 历史交易数量 > 补库提前期补库和调拨汇总数量
- **匹配**: 条件1满足

### 4. 基线异常

- **条件1**: 修改基线ROP < 推荐基线（修改基线ROP存在时）
- **条件2**: 历史交易数量 > 原始基线
- **条件3**: 原始基线=0 且 历史交易数量=0 且 网容>0
- **条件4**: 修改基线ROP为空，且原始基线 < 推荐基线，且历史交易数量=0，且网容>0
- **匹配**: 条件1或条件2或条件3或条件4满足

### 5. 计划参数异常

- **条件1**: 最终补库提前期 < 计算补库提前期
- **匹配**: 条件1满足

### 6. 补库供应不及时

- **条件1**: 补库在途时间（天） > 最终补库提前期
- **条件2**: 调拨在途时间（天） > 最终补库提前期
- **匹配**: 条件1或条件2满足
- **注意**: 补库在途时间、调拨在途时间格式为"数字d"（天）

### 7. 责任库房异常

- **条件1**: 责任库房预测物流时长（天） > SLA承诺时间（天）
- **匹配**: 条件1满足
- **注意**: 责任库房预测物流时长、SLA承诺时间格式为"数字h"（小时）

### 8. 替代交付异常

- **条件1**: 单一替代库存汇总 < 总欠料数量
- **条件2**: 组合替代关系非空，且组合替代库存汇总为空，或组合替代库存无法覆盖组合替代需求
- **条件3**: 当组合替代关系为空[]时，若单一替代库存汇总 < 总欠料数量，且 M2=0，且总欠料数量=1，且不属于补库供应不及时，且历史交易数量=0，且最终补库提前期>=60，也标记为替代交付异常
- **匹配**: (条件1且条件2) 或 条件3满足
- **注意1**: 条件2不再把“存在组合替代库存数据”直接视为库存充足，而是逐项检查是否满足需求
- **注意2**: 条件3是通过训练集误差归纳、再用测试集验证后保留的保守兜底规则

| 字段名            | 说明            | 数据类型           |
| -------------- | ------------- | -------------- |
| 网容             | 网容数据          | 数值             |
| M2             | 当月用量          | 数值             |
| M3-M13最大值      | 近11个月用量最大值    | 数值             |
| 历史交易数量         | 历史交易数量        | 数值             |
| 原始基线           | 原始基线值         | 数值             |
| 补库提前期补库和调拨汇总数量 | 补库提前期内补库和调拨汇总 | 数值             |
| 修改基线ROP        | 人工修改的基线ROP    | 数值（可为空）        |
| 推荐基线           | 系统推荐基线        | 数值             |
| 最终补库提前期        | 系统应用的补库提前期    | 数值             |
| 计算补库提前期        | 系统计算的补库提前期    | 数值             |

| 总欠料数量          | 总欠料数量         | 数值             |
| 组合替代关系         | 组合替代关系JSON    | 字符串（如"[]"）     |
| 组合替代库存汇总       | 组合替代库存汇总      | 字符串或列表结构       |
| 组合替代需求数量       | 组合替代需求数量      | 字符串或列表结构       |

## 异常处理

- 若所有L2标签都不匹配，结果为 `- 未匹配到分支`
- 多个标签匹配时，用 `、` 连接

## 已知数据问题

验证过程中仍发现示例数据存在以下不一致问题：

1. **相同数据不同标签**: 存在特征相同但标注不同的样本
2. **替代交付异常不一致**: 组合替代关系=[]时，是否标注替代交付异常并不完全稳定
3. **部分异常缺乏外显证据**: 个别 `基线异常`、`补库供应不及时` 样本很难从现有字段中推断

建议在使用前检查数据标注的准确性。
